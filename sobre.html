Sobre o projeto

Esse site é um projeto pessoal. Foi criado com o intuito de colocar em prática os meus conhecimentos adquiridos nos crusos de Programação (lógica de programação em Javascript e HTML), Desenvolvimento Front-end (HTML5 e CSS3) e Javascript para Back-end (fundamentos do Javascript, algoritmos com Javascript e Node.js).

A página foi construída do zero, foram usadas apenas a linguagem HTML de marcação, a linguagem CSS de estilos e a linguagem Javascript de programação. A animação principal, que muda a posição do gradiente das cores lhe dando um efeito de movimento, foi feita com a utilização de @keyframes do CSS. A linguagem Javascript foi utilizada para reconhecer as cores escolhidas pelo usuário e usá-las para substituir as cores padrão do efeito. O código pode ser encontrado no meu Github.

Sou formado em Ecologia e Análise Ambiental pela Universidade Federal de Goiás e Pós-graduado em Gestão de Recursos Hídricos pela UNINTER. Porém, recentemente descobri minha paixão pela programação e tenho dado o meu melhor para conseguir construir minha carreira nessa área. Até o momento da producção desse projeto, possuo cerca de 150 horas de cursos em programação aplicada no desenvolvimento web. Além dos projetos feitos com o acompanhamento dos professores durante as aulas, esse é o primeiro projeto que faço por conta própria, sem o auxílio de colegas do curso. A medida que vou aprendendo a usar mais ferramentas, criarei mais projetos, cada vez mais elaborados.

Obrigado por ter chegado aqui. Até breve!.
